# Chief Preferences

- **Agent Invocation Prefix**: The Chief prefers using the `/` prefix as the default for invoking all agents (e.g., `/NEXUS`, `/SENTINEL`, `/GUARDIAN`, `/ZEUS`).
- **Command Mappings**: Map short-form commands (e.g., `/ATLAS`) globally to their functional agent invocation (e.g., `/workspace.ATLAS`) to simplify interaction. `/COMMAND` is an alias for `/workspace.COMMAND`.
- **Authority**: Chief (The Kingmaker) is the absolute authority.
